# STAR - Lyrics Search Utility

Genius API를 활용하여 노래의 가사를 탐색하고 관리하는 데스크탑 기반의 가사 검색 솔루션입니다. 복잡한 설정 없이 노래 제목만으로 가사 정보를 즉시 불러오며, 직관적인 인터페이스를 통해 사용자의 편의성을 최적화했습니다.

### ⚙️ 핵심 기능 및 특징

**1. 간편한 데이터 검색**
- 노래 제목 입력을 통해 Genius 서버에서 아티스트, 앨범 정보 및 가사 데이터를 실시간으로 추출합니다.
- 복잡한 API 인증 과정 없이 즉시 검색 기능을 이용할 수 있도록 토큰 시스템이 내장되어 있습니다.

**2. 고도화된 UI/UX**
- 음악 감상 경험을 극대화하기 위해 '바이닐 디스크(Vinyl Disc)' 애니메이션을 적용하여 심미적 요소를 강화했습니다.
- 사용자 환경에 맞춘 5가지 테마 모드(Default, Moonlight, Dark, White, Rainy Ashville)를 제공하여 시각적 피로도를 최소화합니다.

**3. 사용자 중심의 편의 기능**
- **클립보드 복사**: 단 한 번의 클릭으로 검색된 가사를 클립보드로 복사할 수 있습니다.
- **텍스트 저장**: 검색된 가사를 `.txt` 형식으로 PC 로컬 저장소에 즉시 기록할 수 있습니다.
- **가독성 조절**: 텍스트의 가독성을 높이기 위해 실시간 글자 크기 조절(A+/A-) 기능을 지원합니다.
- **오류 대응**: 네트워크 장애나 환경 변화에 대비한 HTTP 폴백 서버 기능을 내장하여 안정적인 구동을 보장합니다.

**4. 최적화된 성능**
- Python `pywebview` 환경에서 실행되어 별도의 브라우저 설치 과정 없이 독립된 창으로 가볍게 구동됩니다.

---

### 🚀 이용 방법
1. Release 페이지에서 최신 버전의 `.exe` 파일을 다운로드합니다.
2. 실행 후 검색창에 제목을 입력하여 결과를 확인합니다.
3. 설정 메뉴를 통해 취향에 맞는 UI 테마를 선택하여 사용하십시오.
