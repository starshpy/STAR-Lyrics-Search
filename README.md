# 🌟 STAR - 노래 가사 검색기

Genius API 기반의 빠르고 간편한 가사 검색 데스크탑 애플리케이션입니다.

---

### 🚀 주요 기능
• **즉시 검색**: 노래 제목만으로 가사와 앨범 정보를 빠르게 조회합니다.
• **고급 UI/UX**: 감성적인 'Vinyl Disc' 애니메이션과 5가지 커스텀 테마를 지원합니다.
• **편의 기능**:
  - 원클릭 클립보드 복사
  - 로컬 .txt 파일 저장
  - 가독성을 위한 실시간 글자 크기 조절
• **경량화**: 별도의 설치 과정이 필요 없는 `.exe` 배포 방식입니다.

---

### 🎨 테마 모드
`설정(톱니바퀴)`에서 원하는 분위기를 선택하세요.
> `Default` | `Moonlight` | `Dark` | `White` | `Rainy Ashville`

---

### 📥 다운로드 및 실행
1. [Release 페이지](#)에서 최신 `STAR.exe` 파일을 다운로드합니다.
2. 파일을 실행하여 바로 사용합니다.
   * 별도의 API 키 설정이 필요 없는 **고정 토큰 방식**으로 즉시 이용 가능합니다.

---

### 🛠 기술 스택
**Backend**: Python, `pywebview`, `lyricsgenius`
**Frontend**: HTML5, CSS3, Vanilla JS
